# /zhomes reload \<type>

{% hint style="info" %}
"\*" means every file in the directory
{% endhint %}

| Reload Type | Description                                                             | Files                                       |
| ----------- | ----------------------------------------------------------------------- | ------------------------------------------- |
| all         | Reloads all [**plugin files**](../files/)                               | \*                                          |
| commands    | Reloads all plugin commands in [**config.yml**](../files/config.yml.md) | [**config.yml**](../files/config.yml.md)    |
| config      | oads the whole [**config.yml**](../files/config.yml.md)                 | [**config.yml**](../files/config.yml.md)    |
| languages   | Reloads all plugin languages                                            | [**languages/**](../files/languages/)**\*** |
